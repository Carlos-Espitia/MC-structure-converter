To use S2CB (Schematic To Command Block), make sure you have Java 1.8 or higher installed. Extract
all files in this archive to the same directory and then double click on the S2CBv116.jar file.  
This will run Schematic2CommandBlock, and display the main screen with instructions on how to use 
the program.  Upon processing a structure, the instructions will be replaced with the commands to 
reproduce the structure, you have to run the program again to re-show the instructions.
